## 1. Product Overview
专业求职简历网页，用于展示郑鑫源的个人职业信息、技能、实习经历及项目成果，帮助招聘方快速了解候选人核心竞争力。
- 目标用户：嵌入式软件开发/物联网开发/后端开发岗位招聘人员
- 核心价值：以简洁专业的设计呈现完整技术链路实践经验，突出实习成果与项目亮点

## 2. Core Features

### 2.1 Feature Module
1. **首页简历**：个人信息概览、专业技能展示、教育背景、实习经历、项目经历、获奖情况、自我评价

### 2.2 Page Details
| Page Name | Module Name | Feature description |
|-----------|-------------|---------------------|
| 首页简历 | 个人信息区 | 展示姓名、照片、联系方式、意向岗位等基本信息 |
| 首页简历 | 专业技能区 | 按类别展示嵌入式开发、编程语言、通信协议、全栈开发等技能 |
| 首页简历 | 教育背景区 | 展示院校、专业、学历、主修课程 |
| 首页简历 | 实习经历区 | 展示实习公司、时间、职责及量化成果 |
| 首页简历 | 项目经历区 | 展示项目名称、技术栈、职责及关键成果 |
| 首页简历 | 获奖情况区 | 展示证书/奖项名称及颁发单位 |
| 首页简历 | 自我评价区 | 展示个人职业总结与发展目标 |

## 3. Core Process
用户打开页面 → 浏览个人信息 → 查看专业技能 → 了解实习经历 → 查看项目成果 → 阅读自我评价 → 联系候选人

## 4. User Interface Design

### 4.1 Design Style
- **主色调**：深蓝色(#1e3a5f)搭配科技蓝(#007bff)，体现专业稳重感
- **辅助色**：浅灰(#f8f9fa)作为背景，深灰(#343a40)作为文字
- **按钮风格**：圆角矩形，蓝色渐变，悬停时有阴影效果
- **字体**：标题使用思源黑体粗体，正文使用思源黑体常规
- **布局风格**：单页卡片式布局，左侧个人信息，右侧详细内容
- **图标风格**：简洁线性图标，蓝色调

### 4.2 Page Design Overview
| Page Name | Module Name | UI Elements |
|-----------|-------------|-------------|
| 首页简历 | 个人信息区 | 证件照圆形展示、姓名大字标题、联系方式图标列表、意向岗位标签 |
| 首页简历 | 专业技能区 | 分类卡片、技能标签云、进度条展示熟练度 |
| 首页简历 | 教育背景区 | 时间线布局、学校信息卡片 |
| 首页简历 | 实习经历区 | 公司Logo占位、时间轴、职责描述要点、量化成果高亮 |
| 首页简历 | 项目经历区 | 项目卡片、技术栈标签、项目亮点列表 |
| 首页简历 | 获奖情况区 | 证书图标、奖项名称、颁发单位 |
| 首页简历 | 自我评价区 | 段落文本、核心优势标签 |

### 4.3 Responsiveness
- **桌面端(>1200px)**：双栏布局，左侧个人信息固定，右侧滚动内容
- **平板端(768px-1200px)**：单栏自适应布局
- **移动端(<768px)**：单栏垂直布局，字体缩小，间距紧凑

### 4.4 交互设计
- 技能卡片悬停放大效果
- 项目经历展开/收起动画
- 平滑滚动导航
- 打印友好样式
